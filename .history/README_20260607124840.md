# 🚀 FutureCareer AI: Futuristic Resume Fixer & Skill Mentor

**FutureCareer AI** is a full-stack, AI-powered career companion designed to analyze resume details, provide critical text optimization, highlight skill gaps, and generate structured roadmaps for professional growth. 

Featuring a sleek, high-contrast cyberpunk interface inspired by the palette in `image_75bba7.png`, this project integrates an **HTML5/CSS3 frontend**, a lightweight **PHP backend processing layer**, and a **fine-tuned open-source Large Language Model (LLM)** via Hugging Face.

---

The interface is strictly designed around the futuristic color scheme extracted from `image_75bba7.png`:

---

## 🛠️ Tech Stack & Ecosystem

### 1. Frontend
*   **HTML5 & CSS3:** Vanilla structural semantic markup with an advanced glassmorphic terminal theme, neon text shadows, and responsive grid panels.

### 2. Backend
*   **PHP:** Handles server-side request management, inputs validation, formatting, and secure communication with the local AI python inference microservice via cURL / Localhost sockets.

### 3. AI Engine & Fine-Tuning
*   **Base Model:** `Qwen/Qwen2.5-1.5B-Instruct` (or `Llama-3.2-1B-Instruct`)
*   **Fine-Tuning Paradigm:** Supervised Fine-Tuning (SFT) using Parameter-Efficient Fine-Tuning (**PEFT**) via Low-Rank Adaptation (**LoRA**).
*   **Quantization:** 4-bit NormalFloat (`nf4`) via `bitsandbytes` to enable local deployment and fast iteration on consumer-grade or free cloud GPUs (e.g., Google Colab T4).
*   **Libraries:** `transformers`, `datasets`, `peft`, `trl`, `accelerate`.

🔄 Execution Pipeline Workflow
Plaintext
[User UI Panel] ──(Inputs Resume)──> [process.php] ──(Sanitizes)──> [api_client.php]
                                                                          │
[User UI Panel] <──(JSON Feedback)── [process.php] <──(Response)─── [inference.py] 
Input: The user inputs their current skills, profile, and target career path into the terminal interface.

Routing: The form submits to process.php. The backend extracts and wraps the data into a standardized JSON structure.

Inference: api_client.php forwards the text packet to inference.py (which hosts the fine-tuned LoRA weights).

Generation: The model generates actionable, highly contextual resume adaptations and technical next steps.

Render: The backend receives the markdown output, wraps it securely, and pushes it back to the client UI.

🚀 Setup & Installation
Prerequisites
Web Server: Apache/Nginx environment with PHP 8.x installed (e.g., XAMPP, LocalWP, or Docker setup).

Python Environment: Python 3.10+ with CUDA support (if training/running inference locally).

Step-by-Step Installation
Clone and Place Repository:
Move the career-mentor-ai directory into your server's root folder (e.g., htdocs or /var/www/html/).

Install Python Dependencies:
Navigate to the ai_engine folder and initialize your environment:

Bash
   pip install transformers datasets trl peft bitsandbytes accelerate flask
Run Fine-Tuning:
Execute the script to inject domain knowledge into your base LLM:

Bash
   python ai_engine/fine_tune.py
Spin up the AI Microservice:
Start the local background inference engine server:

Bash
   python ai_engine/inference.py
Access UI:
Open your browser and navigate to http://localhost/career-mentor-ai/frontend/index.html.
