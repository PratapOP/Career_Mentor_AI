# 🚀 FutureCareer AI: Futuristic Resume Fixer & Skill Mentor

**FutureCareer AI** is a full-stack, AI-powered career companion designed to analyze resume details, provide critical text optimization, highlight skill gaps, and generate structured roadmaps for professional growth. 

Featuring a sleek, high-contrast cyberpunk interface inspired by the palette in `image_75bba7.png`, this project integrates an **HTML5/CSS3 frontend**, a lightweight **PHP backend processing layer**, and a **fine-tuned open-source Large Language Model (LLM)** via Hugging Face.

---

## 🎨 Design System & Palette
The interface is strictly designed around the futuristic color scheme extracted from `image_75bba7.png`:
*   **Primary Background:** Cyberpunk Deep Black (`#0D0D0D`)
*   **Accent/Glow Color:** Neon Magenta / Raspberry Crimson (`#C22755`)
*   **Text & Sub-elements:** Synth Grey (`#E2E2E2`) and High-Contrast Platinum (`#EDEDED`)

---

## 🛠️ Tech Stack & Ecosystem

### 1. Frontend
*   **HTML5 & CSS3:** Vanilla structural semantic markup with an advanced glassmorphic terminal theme, neon text shadows, and responsive grid panels.

### 2. Backend
*   **PHP:** Handles server-side request management, inputs validation, formatting, and secure communication with the local AI python inference microservice via cURL / Localhost sockets.

### 3. AI Engine & Fine-Tuning
*   **Base Model:** `Qwen/Qwen2.5-1.5B-Instruct` (or `Llama-3.2-1B-Instruct`)
*   **Fine-Tuning Paradigm:** Supervised Fine-Tuning (SFT) using Parameter-Efficient Fine-Tuning (**PEFT**) via Low-Rank Adaptation (**LoRA**).
*   **Quantization:** 4-bit NormalFloat (`nf4`) via `bitsandbytes` to enable local deployment and fast iteration on consumer-grade or free cloud GPUs (e.g., Google Colab T4).
*   **Libraries:** `transformers`, `datasets`, `peft`, `trl`, `accelerate`.
